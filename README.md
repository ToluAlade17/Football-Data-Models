# Football Data Models

This repository is my practice space for football modelling and learning Python through football data analysis.

The first project in this space is `dangerous carries`, built using StatsBomb open data. I am using it to practise how to:

- load and inspect football event data
- clean and reshape raw data into analysis-ready tables
- engineer useful football features from carries and possessions
- explore how carries relate to shot creation and attacking threat
- build a reproducible workflow that can support future modelling work

The goal is not just to produce one final output, but to understand each step properly as I go and build the habits I would use in a football data analyst or football data science role.

Over time, I want this repository to expand beyond dangerous carries into more ambitious models, new ideas, and hypothesis testing across different football questions.
