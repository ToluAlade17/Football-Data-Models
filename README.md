# Football Data Models

This repository is my practice space for learning Python through football data analysis.

I am using it to work with StatsBomb open data and build a small end-to-end analysis project around dangerous carries. My aim is to learn how to:

- load and inspect football event data
- clean and reshape raw data into analysis-ready tables
- engineer useful football features from carries and possessions
- build a reproducible workflow that can later be used for modelling
- explore ideas related to dangerous carries and expected threat

The main goal of this project is not just to produce a final model, but to understand each step properly as I go. I want to use this repo to practise the kind of workflow I might use in a football data analyst or football data science role.

Over time, I want this project to grow into a structured pipeline that starts with raw event data and ends with a simple football analysis model that helps identify carries that increase attacking threat.
